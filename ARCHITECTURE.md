# AI Engineer Architecture

## Modules

### Brain

Understands the user request.

### Planner

Creates execution plans.

### Knowledge Engine

Searches Unity documentation.

### Vision Engine

Analyzes Unity Scene.

### Action Engine

Creates executable commands.

### Unity Plugin

Executes commands inside Unity.
